Esse é o desafio da Trilha de CSS da DIO! Nela, foi construido uma Landing Page com HTML e CSS, colocando em prática os fundamentos do CSS,
as propriedades básicas da linguagem de estilização, além de trabalhar com as unidades de medidas relativas e absolutas que foram aprendidas ao longo da trilha.
